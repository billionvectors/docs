## 2024-09-27: First Release

- This is the first official release of **asimplevectors**, a high-performance vector database optimized for retrieval-augmented generation (RAG).
- Key features include:
  - **Raft-based Clustering**: Reliable and fast clustering using the Raft consensus protocol.
  - **Support for Dense and Sparse Vectors**: Efficiently manages both dense and sparse vector types.
  - **SQLite3 Integration**: Simple and effective data management using SQLite3.
  - **Built-in Key-Value Storage**: Supports the storage of original document data alongside vector data.
  - **Version Control and Snapshots**: Version management with A/B testing capabilities and snapshot support for data backups.

Stay tuned for future updates and feature improvements!
